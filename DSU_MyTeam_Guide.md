## 1. 프로젝트 개요
- **이름**: DSU My_team (동서대학교 MYDEX 특화 매칭 시스템)
- **핵심 가치**: 24시간 가동, 72시간 데이터 휘발성(TTL), AI 기반 적합도 분석, 전공/학번 밸런스 매칭.
- **주요 UI**: 카드 버튼 기반 대시보드, 다중 선택 이모지 특기 카드, 신청 Modal.
- **데이터베이스**: Google Sheets (메인 연동) + SQLite (로컬 캐시)

  - 구글 시트에서 수동으로 입력한 신청 데이터가 봇의 매칭 시스템에 정상 인식되도록 처리.

## 4. 전체 파일 구조 (`C:\Users\PC-1\.gemini\antigravity\scratch\DSU_MyTeam`)
- `main.py`: 봇 진입점
- `config.py`: 설정 관리
- `.env`: 디스코드 봇 토큰 및 구글 시트 ID 저장
- `credentials.json`: 구글 서비스 계정 인증 키
- `database/db_manager.py`: 로컬 DB 쿼리 로직
- `tasks/scheduler.py`: 72시간 만료 처리(로컬 + 시트) 스케줄러
- `cogs/`: 봇 기능 모듈 (menu, application, team_room, random_match, group_apply, admin)
- `services/gsheet_service.py`: 구글 시트 연동 로직
- `bot.log`: 봇 디버깅 로그
